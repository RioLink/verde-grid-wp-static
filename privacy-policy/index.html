Політика конфіденційності

1. Загальні положення

Ця Політика конфіденційності визначає порядок збору, використання, зберігання, обробки та захисту інформації, яку Verde Grid Advisory LLC може отримувати під час використання користувачами вебсайту, цифрових сервісів, ESG-платформ, форм зворотного зв’язку, інтеграцій та інших технологічних рішень компанії.

Використовуючи сайт Verde Grid або взаємодіючи з будь-якими цифровими сервісами компанії, користувач підтверджує ознайомлення з цією Політикою та погоджується з умовами обробки інформації.

Ми прагнемо забезпечити прозорий та відповідальний підхід до конфіденційності, цифрової безпеки та захисту даних у межах нашої операційної та ESG-інфраструктури.

2. Які дані ми можемо збирати

Verde Grid може збирати різні категорії інформації залежно від типу взаємодії користувача із сервісами компанії.

2.1 Контактні дані

  • ім’я та прізвище;
  • номер телефону;
  • адреса електронної пошти;
  • назва компанії;
  • посада або роль у компанії.

2.2 Технічна інформація

  • IP-адреса;
  • тип браузера;
  • тип пристрою;
  • операційна система;
  • мовні налаштування;
  • час взаємодії із сайтом;
  • технічні журнали подій.

2.3 ESG та операційні дані

  • дані енергоспоживання;
  • показники водних ресурсів;
  • інформація про відходи;
  • дані датчиків та BMS-систем;
  • операційні звіти;
  • аналітичні метрики.

Частина даних може надаватися користувачем добровільно через форми, заявки або інтеграції корпоративних систем.

3. Для чого ми використовуємо інформацію

Verde Grid використовує дані виключно в межах законних операційних, технічних та аналітичних процесів.

Інформація може використовуватися для:

  • роботи цифрових сервісів;
  • надання ESG-аналітики;
  • створення операційних дашбордів;
  • зв’язку з користувачами;
  • обробки заявок;
  • поліпшення UX/UI;
  • захисту цифрової інфраструктури;
  • аналітики навантаження;
  • виявлення технічних помилок;
  • розвитку нових функцій.

Ми не використовуємо персональні дані для незаконного профілювання або автоматизованого прийняття рішень, що можуть мати юридичні наслідки для користувачів.

4. ESG-аналітика та операційні дані

Частина сервісів Verde Grid пов’язана зі збором та аналізом операційних, екологічних та ESG-даних.

Такі дані можуть включати:

  • показники споживання енергії;
  • показники водоспоживання;
  • дані про відходи;
  • дані IoT-пристроїв;
  • показники навантаження об’єктів;
  • інформацію ERP та BMS систем.

ESG-дані використовуються для формування звітності, аналітики, прогнозування та операційної оптимізації.

Verde Grid прагне мінімізувати обсяг персональних даних у межах ESG-аналітики та використовувати агреговану або технічну інформацію, коли це можливо.

5. Cookie та технічні технології

Для забезпечення стабільної роботи сайту Verde Grid використовує cookie, локальне сховище браузера, технічні журнали та подібні технології.

Такі інструменти допомагають:

  • зберігати мовні налаштування;
  • аналізувати роботу сторінок;
  • захищати форми від спаму;
  • підтримувати безпеку сесій;
  • покращувати продуктивність сайту;
  • оцінювати навантаження серверів.

Детальна інформація щодо використання cookie описана в окремій Cookie Policy.

6. Передача та розкриття інформації

Verde Grid не продає персональні дані користувачів третім особам.

Передача інформації можлива лише у випадках:

  • виконання договірних зобов’язань;
  • роботи технічних інтеграцій;
  • використання хмарної інфраструктури;
  • виконання вимог законодавства;
  • захисту цифрової безпеки;
  • обробки ESG-даних для клієнтів.

Усі партнери та постачальники сервісів зобов’язані дотримуватись належного рівня захисту інформації.

7. Зберігання та термін обробки даних

Дані можуть зберігатися протягом періоду, необхідного для:

  • роботи сервісів;
  • аналітики та звітності;
  • виконання договірних умов;
  • технічного моніторингу;
  • забезпечення безпеки;
  • дотримання законодавчих вимог.

Після завершення необхідного строку дані можуть бути видалені, анонімізовані або архівовані відповідно до внутрішніх процедур.

8. Захист інформації

Verde Grid застосовує організаційні та технічні заходи для захисту цифрової інфраструктури та даних користувачів.

Заходи безпеки можуть включати:

  • шифрування даних;
  • контроль доступу;
  • мережевий моніторинг;
  • захист API;
  • журнали активності;
  • системи виявлення аномалій;
  • резервне копіювання.

Незважаючи на використання сучасних технологій безпеки, жодна система не може гарантувати абсолютний захист від усіх кіберзагроз.

9. Права користувачів

Користувач має право:

  • отримувати інформацію про обробку даних;
  • уточнювати або оновлювати інформацію;
  • відкликати згоду на обробку даних;
  • вимагати видалення інформації;
  • обмежувати окремі види обробки;
  • надсилати звернення щодо конфіденційності.

Частина прав може бути обмежена у випадках, передбачених чинним законодавством або договірними зобов’язаннями.

10. Міжнародна передача інформації

Окремі технічні сервіси Verde Grid можуть використовувати хмарну інфраструктуру або дата-центри, розташовані за межами України.

У випадку міжнародної передачі інформації компанія прагне забезпечувати належний рівень захисту даних відповідно до сучасних стандартів цифрової безпеки та конфіденційності.

11. Дані неповнолітніх

Сервіси Verde Grid не призначені для осіб молодше 18 років.

Компанія свідомо не збирає персональні дані неповнолітніх без відповідного дозволу законних представників.

12. Сторонні сервіси та інтеграції

Для роботи цифрових сервісів Verde Grid можуть використовуватися сторонні технології та інтеграції, включаючи:

  • хмарні платформи;
  • аналітичні сервіси;
  • системи моніторингу;
  • API сторонніх платформ;
  • карти та геолокаційні сервіси;
  • сервіси кібербезпеки.

Такі сервіси можуть працювати відповідно до власних політик конфіденційності та технічних стандартів.

13. Оновлення Політики конфіденційності

Verde Grid Advisory LLC залишає за собою право змінювати або оновлювати цю Політику конфіденційності у будь-який час.

Оновлена редакція набирає чинності з моменту її публікації на сайті, якщо інше не передбачено додатковими умовами.

Рекомендуємо періодично переглядати цю сторінку для ознайомлення з актуальною редакцією Політики.
